# AI Legal Advisor Repository

This repository contains the source of truth, metadata, templates, and AI integration services for the Vietnamese Legal Advisory Assistant.

## Directory Structure
- `knowledge_base/`: Raw Markdown/PDF versions of official legal documents.
- `metadata/`: Schemas, registries, and mappings to govern the legal documents.
- `prompts/`: Version-controlled prompts for Gemini and Spring AI agents.
- `templates/`: Structured JSON/Markdown templates for output generation.
- `rag_pipeline/`: Python scripts for data ingestion into Qdrant.
- `services/`: Spring Boot Java microservices for orchestrating the AI workflow.
- `qdrant_config/`: Configurations for the Vector Database.

## Getting Started
Please refer to the READMEs in each sub-directory for specific instructions on local development and deployment.
